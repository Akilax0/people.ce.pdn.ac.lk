Unit tests for people.ce.pdn.ac.lk
How to run:
Download the chromedriver from https://chromedriver.chromium.org/downloads and put it in the webdriver folder.
Put the server URL in the SERVER_URL variable. (a_config_test.py file)
Run this script from the command line.
    'make test' 
    or 
    'python3 -m unittest discover'
