--- layout: documentation permalink: /documentation/edit-student-page/ question: How to edit a student profile page ? index: 100 ---
  1. Go to the student profile page that needs to be edited
    • Manually navigate to the page
    • Or, Use the search page to find it
      • On the search page {% include documentation_image.html url_image='/images/docs/search_reg_no.PNG' text='your' width='1706' height='553'%} your registration number
    • If your profile page is missing, please click here

  2. Click "Edit this page on GitHub" link at the{% include documentation_image.html url_image='/images/docs/edit_this_on_github.png' text='bottom' width='1348' height='772' %} of the page.

  3. Click on "Edit" {% include documentation_image.html url_image='/images/docs/click_edit_inside_github.png' text='icon' width='1904' height='802'%} .

  4. Make your changes to the file

    Listed below are the variables that are used to edit your profile page.
    Note: If you want to keep a value as empty. Type # next to the variable
    Click {% include documentation_image.html url_image='/images/docs/studentDetails_description.png' text='here' width='1643' height='870'%} to view the locations of those variables in the page
    Variable Description
    layout Used internally. Do not change.
    permalink The link to your profile page. Do not change
    reg_no Your registration number
    batch Your batch number. Used internally. Do not change
    full_name Your full name that is displayed in the page
    name_with_initials This is displayed in the {% include documentation_image.html url_image='/images/docs/name_with_intials.png' text='batch' width='1677' height='762'%} page.
    preferred_short_name Displayed under the photo
    preferred_long_name Displayed in the page
    honorific Mr. / Miss. ... etc
    email_faculty @eng.pdn.ac.lk email address
    email_personal Your personal email address
    location Where you are from
    url_cv Link to your CV
    url_website Link to your website
    url_linkedin Link to your LinkedIn profile
    url_github Link to your GitHub profile
    url_researchgate Link to your ResearchGate profile
    url_twitter Link to your Twitter profile
    interests List your interests in comma seperated format. Eg: IoT, Machine Learning, Embedded Systems
    projects_done This will be filed automatically. Do not change
    image_url This is the image in your profile page. Do not change for now. If you want to change the profile picture. Follow these instructions seperately.
    Example: 
                ---
layout: studentDetails
permalink: "/students/e18/098/"

reg_no: E/18/098
batch: E18

full_name: Kurukulasuriya Anton Ishan Fernando
name_with_initials: K.A.I. Fernando
preferred_short_name: Ishan
preferred_long_name: Ishan Fernando
honorific: Mr.

email_faculty: e18098@eng.pdn.ac.lk
email_personal: #

location: Negombo

url_cv: #
url_website: #
url_linkedin: #
url_github: https://github.com/ishanfdo18098
url_facebook: https://facebook.com/ishanfdo
url_researchgate: #
url_twitter: #

interests: interest1, interest2
projects_done: []
image_url: "images/students/e18/e18098.jpg"
---

  5. Scroll down then click "Propose Changes" {% include documentation_image.html url_image='/images/docs/propose_changes.png' text='button' width='715' height='403'%}.

  6. Click "Create Pull Request" {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='button' width='1848' height='686'%}.

  7. Click "Create Pull Request" {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='button' width='1399' height='701'%} again.

  8. Now an admin will approve these changes within 48hrs.
