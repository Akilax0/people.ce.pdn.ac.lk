--- layout: documentation permalink: /documentation/change-add-student-profile-pic/ question: How to change/add a new student profile image ? index: 102 ---
  1. Go to the GitHub repository of this site

  2. Click {% include documentation_image.html url_image='/images/docs/click_fork.png' text='"Fork"' width='749' height='766' %}

  3. Click {% include documentation_image.html url_image='/images/docs/click_images.PNG' text='"images"' width='678' height='900' %}

  4. Click{% include documentation_image.html url_image='/images/docs/click_students.PNG' text='"students"' width='814' height='795' %}

  5. Click on your batch number

  6. Click on {% include documentation_image.html url_image='/images/docs/upload_files.PNG' text='"Upload files"' width='1260' height='723' %}

  7. Drag and drop your files to the {% include documentation_image.html url_image='/images/docs/drag_and_drop.PNG' text='given place' width='1777' height='833' %}
    • Give the file a descriptive name. Eg E18576.jpg, IshanFernando.jpg etc.
    • Only upload square photos. If you don't have square photos, Use this site to crop your photo to 1:1 aspect ratio

  8. Click {% include documentation_image.html url_image='/images/docs/commit_changes.PNG' text='"Commit changes"' width='911' height='591' %}

  9. Click on {% include documentation_image.html url_image='/images/docs/click_on_commit_msg.PNG' text='"Add files via upload"' width='1038' height='646' %}

  10. Copy the new file path by clicking on the copy {% include documentation_image.html url_image='/images/docs/copy_new_file_path.PNG' text='button' width='844' height='817' %}

  11. Now go to the student page that you want to change the profile picture of.
    • Click {% include documentation_image.html url_image='/images/docs/pages.PNG' text='pages' width='856' height='855' %}
    • Click {% include documentation_image.html url_image='/images/docs/students.PNG' text='students' width='825' height='790' %}
    • Click your batch number
    • Click on your registration number file name. Eg. e18568.html

  12. Click on{% include documentation_image.html url_image='/images/docs/edit_file.PNG' text='edit' width='1817' height='768' %}

  13. Add the end of that page, change the path of the "image_url" to the new path that was copied in step 10
    • Example, 
      image_url: "images/students/e18/e18098.jpg"

  14. Click {% include documentation_image.html url_image='/images/docs/open_pull_request.PNG' text='"Contribute"' width='1284' height='781' %} then click "Open pull request"

  15. Click {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='"Create pull request"' width='1848' height='686' %}

  16. For the {% include documentation_image.html url_image='/images/docs/pull_req_title.PNG' text='Title' width='1412' height='880' %}, write "Update profile image - E18456" replace 18456 with your registration number

  17. Click {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='"Create pull request"' width='1399' height='701' %}

  18. Now an admin will approve these changes within 48hrs.
