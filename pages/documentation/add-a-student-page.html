--- layout: documentation permalink: /documentation/add-student-page/ question: How to add a new student profile page ? index: 140 title: How to add a new student profile page ? --- There are 2 methods of add a new student profile page.
  1. Update using the Google Form
    • This is the easiest method to add a new profile page.
    • E14 and above, use this form.
    • E13 and below, use this form.
    • Note: Do not update using Pull Requests if you fill the google form.
  2. Update using GitHub Pull Request
    • This method is comparatively harder.
    • Follow the steps below to add a new profile page using pull request
Update using GitHub Pull Request
  1. Login to your GitHub account. If you don't have an account. Please sign up.

  2. Go to this link.

  3. Select your {% include documentation_image.html url_image='/images/docs/click_your_batch.PNG' text='batch'%}.

  4. Click "Add File" then {% include documentation_image.html url_image='/images/docs/create_new_file.PNG' text='"Create new File""' %}.

  5. Write the file name in the given textbox.
    • It should be in "e18596.html" format. Please replace "18596" with your registration number.
    • {% include documentation_image.html url_image='/images/docs/file_name.PNG' text='Example' %}

  6. Copy and paste the code below into the {% include documentation_image.html url_image='/images/docs/copy_paste_text.PNG' text='textbox'%} in the opened webpage
    7. ---
layout: studentDetails
permalink: "/students/e18/596/"
title: #

reg_no: E/18/596
batch: E18

department: Computer
current_affiliation: Department of Computer Engineering, University of Peradeniya

full_name: #
name_with_initials: #
preferred_short_name: #
preferred_long_name: #
honorific: #

email_faculty: #
email_personal: #

location: #

url_cv: #
url_website: #
url_linkedin: #
url_github: #
url_facebook: #
url_researchgate: #
url_twitter: #

interests: #

image_url: "images/students/default.jpg"
---
  7. Edit the following lines
    • permalink: Replace 18 with your batch number, and 596 with your reg number. The result should look like if you are E/14/154, 
      permalink: "/students/e14/154/"
    • reg_no: Type your registration number in the given format
    • batch: Type your batch number in the given format
    • from "full_name:" to "projects_done:" please read step 4 of this documentation
    • Do not change "image_url:" for now. It will be done later.

  8. Click "Propose new file" at the {% include documentation_image.html url_image='/images/docs/propose_new_file.PNG' text='end' %} of the page.

  9. Click "Create pull request" {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='button'%}

  10. Click "Create pull request" {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='button'%} again

  11. Now an admin will approve these changes within 48hrs.

  12. After it its approved, Please go to this page to add your profile picture.
